==============================================
  OPENGRANT - How to Use
  Find grants, hackathons & funding for your
  open source GitHub project - FREE!
==============================================

QUICK START (3 steps):
-----------------------
1. Run SETUP.bat  (only once, first time)
2. Run START.bat  (every time you want to use it)
3. Website opens at http://localhost:5173


STEP 1 - GET YOUR FREE API KEY:
--------------------------------
1. Go to https://console.groq.com
2. Sign up for free (no credit card)
3. Click "Create API Key"
4. Copy the key (starts with gsk_...)
5. Open backend\.env file
6. Replace "your_api_key_here" with your key
7. Save the file


STEP 2 - FIND FUNDING FOR YOUR REPO:
--------------------------------------
1. Go to http://localhost:5173
2. Paste your GitHub repo URL
   Example: https://github.com/your-name/your-project
3. Click "Find Funding"
4. Wait 30-60 seconds for AI analysis
5. See your matched grants ranked by fit!


FEATURES GUIDE:
---------------
HOME PAGE
  - Paste any public GitHub repo URL
  - Works with any language (Python, JS, Rust, etc.)
  - Even small/new projects get matches!

RESULTS PAGE
  - Fundability Score: Your project grade (A to F)
  - Matched Grants: Ranked list of best matches
  - Click "Write Application" -> AI writes full grant text
  - Click "Save" -> Adds to your tracker
  - Click "Apply Now" -> Opens grant website

MY TRACKER (/applications)
  - Track all your saved grants
  - Move through stages: Saved > Applied > Won!
  - Add notes to each application

DEADLINES CALENDAR (/calendar)
  - See all grant deadlines
  - Filter by type (grant, hackathon, bounty)
  - Export to Google Calendar (.ics file)

ORG SCANNER (/org)
  - Enter your GitHub org/username
  - Scans ALL your public repos at once
  - Shows fundability grade for each repo

DEPENDENCY MAP (/dependencies)
  - Paste your package.json or requirements.txt
  - See which dependencies need funding support
  - Identify high-risk unfunded packages

TROUBLESHOOTING:
----------------
Q: Website not opening?
A: Make sure both Backend and Frontend windows are running.
   Try: http://localhost:5173 in your browser manually.

Q: "Analysis failed" error?
A: Check your API key in backend\.env is correct.
   Get free key from: https://console.groq.com

Q: Very slow results?
A: Normal - AI is analyzing 25 funding sources for your repo.
   Usually takes 30-60 seconds.

Q: Port already in use error?
A: Close all OPENGRANT windows and try START.bat again.

Q: Want to use a different AI provider?
A: Edit backend\.env:
   - OpenAI:     LLM_BASE_URL=https://api.openai.com/v1
   - Together:   LLM_BASE_URL=https://api.together.xyz/v1
   - OpenRouter: LLM_BASE_URL=https://openrouter.ai/api/v1
